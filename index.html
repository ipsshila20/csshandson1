What is CSS?

  1. CSS stands for Cascading Style Sheets.
  2. CSS describes how HTML elements are to be displayed on screen,paper,or in other media.
  3. CSS is not neccesary CSS saves a lot of work. can control the layout of multiple web pages all at once
  4. External stylesheet are stored in CSS files.

CSS can be added to HTML documents in 3 ways:

  • Inline- by using the style attribute inside HTML elements1
  • Internal- by using a style element in the head section
  • External- by using a link element to link to an external CSS file.
  • The most common way to add CSS, is to style in the External CSS files, However, in this
    tutorial we will use inline and internal styles,     because this is easier to demonstrate, and easier for you to try it yourself.

When comparing CSS class vs Id,the difference is that CSS class applies a style to multiple elements.ID the other hand,applies a style to one unique element.ID is also
special in that you can use a special URL to link directly to and it's used by JavaScript.

In css, selectors are used to target a specific elements or range of elements on a wen page. Once an elements has been tregered, a style or set of styles can be applied to the
element.

There is a wide range of selectors available.Two of the most commonly used are class and ID.Both are used to target elements to which a style should be applied.
